# TrafficInsight Assistant

TrafficInsight Assistant is an AI‑powered data analysis tool designed to explore, summarize and visualize traffic data from Flanders.  
The assistant can interpret datasets, generate insights, create visualizations and explain traffic patterns in natural language.

This project combines:

- Python data analysis (pandas, numpy)  
- Data visualization (matplotlib, seaborn)  
- AI‑driven interpretation (LLM‑based assistant)  
- A simple interactive interface (Streamlit)  

The goal is to build a smart, user‑friendly assistant that helps users understand mobility trends, traffic intensity and congestion patterns in a clear and intuitive way.
